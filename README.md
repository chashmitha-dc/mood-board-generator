# 🎨 Mood Board Generator

A simple aesthetic web application that generates random mood boards using quotes, pastel color palettes, and aesthetic icons.

## ✨ Features
- Random aesthetic quotes
- Beautiful pastel color palettes
- Aesthetic emoji icons
- Generate new mood boards with one click

## 🛠 Built With
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Open the project
2. Click **Generate Mood Board**
3. Get a new aesthetic mood board every time
